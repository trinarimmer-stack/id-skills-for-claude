# Skill Combinations for Common ID Projects

Individual skill files work well on their own. 
But the real power of this library comes from 
loading multiple files together — giving Claude 
the full context it needs to support 
complex, multi-faceted design work.

These combinations are tested starting points. 
Adapt them to your project.

---

## How to Use This Guide

For each combination, copy the contents 
of the listed skill files and paste them 
into Claude's context before making your request. 
The more relevant context Claude has, 
the more useful its output will be.

## Loading Multiple Files: Practical Tips

- **Start focused.** Two or three files 
  is usually enough. More context 
  isn't always better — relevance is.

- **Lead with the task.** After pasting 
  the skill files, state your specific 
  request clearly. The more specific 
  the request, the more useful the output.

- **Iterate.** Claude can hold context 
  across a conversation. Load the files 
  once and work through multiple 
  related tasks in the same session.

- **Mix and match.** These combinations 
  are starting points — trust your 
  judgment about what context 
  your specific project needs.
